# React Skeleton

## Start App

Clone repo
```
sudo git clone https://github.com/sandim27/react-skeleton.git
```

Install all dependencies
```
npm install
```


Rebuild node-sass for your OS configuration
```
npm rebuild node-sass
```

Start app
```
npm start
```

## Generate Documentation

```
npm run jsdoc
```

Check docs/index.html


## Testing

Start unit tests:
```
npm test
```


Reports:
```
npm run test-r
```

## Build

Build app
```
npm run build
```
